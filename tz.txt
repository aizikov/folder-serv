Техническое задание на разработку программы "Файловый менеджер"

Общее описание:
Необходимо разработать программу "Файловый менеджер" для работы с файлами в директории. Программа должна предоставлять следующие функции: просмотр списка файлов в директории, удаление файлов, создание пустого файла, переименование файла, редактирование файла, поиск файлов, отображение информации о файле, сортировка файлов.

Функциональные требования:
1. Отображение списка файлов в директории.
2. Удаление выбранного файла из списка.
3. Создание пустого файла.
4. Переименование выбранного файла.
5. Редактирование содержимого выбранного файла.
6. Поиск файлов по заданной маске.
7. Отображение информации о выбранном файле.
8. Сортировка файлов по имени, расширению или времени модификации.

Интерфейс программы:
1. Окно программы должно содержать элементы управления: метка для отображения текущей маски, список для отображения файлов, кнопки для основных операций (удаление, создание, переименование, редактирование, поиск, отображение информации, сортировка).
2. При выборе определенной операции (удаление, создание, переименование и т.д.) должно выполняться соответствующее действие с выбранным файлом.

Технические требования:
1. Программа должна быть разработана на языке Python с использованием библиотеки PyQt5 для создания графического интерфейса.
2. Для работы с файлами необходимо использовать модуль os.
3. Для работы с датой и временем необходимо использовать модуль datetime.

Тестирование:
1. Программа должна быть протестирована на наличие ошибок при выполнении всех основных операций (удаление, создание, переименование и т.д.).
2. Программа должна корректно обрабатывать ситуации, когда выбранного файла не существует или невозможно выполнить запрошенное действие.

Примечания:
1. Программа должна быть реализована в виде класса FileManagerUI, который наследуется от класса QMainWindow.
2. Файлы в директории должны отображаться в виджете QListWidget.
3. При удалении файла необходимо вывести диалоговое окно для подтверждения операции.
4. При редактировании файла необходимо использовать стандартный текстовый редактор.
5. Для поиска файлов необходимо использовать стандартное диалоговое окно для ввода маски.
6. Информация о файле должна содержать его имя, размер, тип и время модификации.
7. Сортировка файлов должна выполняться по выбранному критерию (имени, расширению или времени модификации).
